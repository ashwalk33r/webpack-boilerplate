# setup

### prerequisites

-   **nvm** for _Linux/Mac_ - https://github.com/nvm-sh/nvm#install--update-script
-   **nvm** for _Windows_ - https://github.com/coreybutler/nvm-windows

### configure

run:

`nvm install 14`

`nvm use 14`

`npm install --global yarn`

`yarn install`

### build

`yarn run build`

### run

`yarn run start`

#prettier

-   `yarn run prettier`
-   `yarn run prettier:fix`

#linter

-   `yarn run prettier`
-   `yarn run prettier:fix`
